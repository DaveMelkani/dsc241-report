# DSC 241 – Statistical Models  
## Winter 2026 Final Project

This repository contains the final project for **DSC 241: Statistical Models (Winter 2026)**. The objective of the project is to apply statistical modeling techniques learned in class to solve a practical data analysis problem.

## Project Overview

The project includes:

- A clearly defined **problem statement**
- Detailed **data description** (structure, variables, sampling, response, covariates)
- A complete **statistical analysis workflow**
  - Model specification
  - Estimation and inference
  - Diagnostics and model selection
  - Visualizations and interpretation
- **Conclusions** connecting results to the original objective
- Supplementary analyses in an appendix
- Fully reproducible **code**

## Deliverables

- Final report (10 pages, professional format)
- Reproducible analysis code

## Goal

To demonstrate rigorous statistical reasoning, clear communication, and proper application of statistical modeling techniques in a real-world data context.
